Detailed Description

Block extensions for NSNumber.

Both inspired by and resembling Smalltalk syntax, these utilities allows for iteration of an array in a concise way that saves quite a bit of boilerplate code for performing a task a fixed number of times.

Includes code by the following:

  • Colin T.A. Gray
