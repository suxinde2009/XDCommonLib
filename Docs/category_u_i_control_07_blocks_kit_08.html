- (void) bk_removeEventHandlersForControlEvents: (UIControlEvents)  controlEvents

Removes all control event blocks associated with the given mask of control events.

Do note that, like UIControl, this method will not decompose control events; thus, only a handler matching an exact given bitmask will be removed.

Parameters
controlEventsA bitmask specifying the control events for which the block will be removed.
