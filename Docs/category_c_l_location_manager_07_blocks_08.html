- (id) blocksDelegate
readnonatomicretain

- (CLUpdateAccuracyFilter) updateAccuracyFilter
readwritenonatomicassign

Specifies the minimum update accuracy when using blocks. Client will not be notified of movements of less than the stated value, unless the accuracy has improved. Pass in kCLUpdateAccuracyFilterNone to be notified of all movements. By default, kCLUpdateAccuracyFilterNone is used.

- (CLLocationAgeFilter) updateLocationAgeFilter
readwritenonatomicassign

Specifies the maximum update location age when using blocks. Client will not be notified locations with an age greater than the stated value. Pass in kCLLocationAgeFilterNone to be notified of all movements. By default, kCLLocationAgeFilterNone is used.
