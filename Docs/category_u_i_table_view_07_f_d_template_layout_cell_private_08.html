- (BOOL) fd_precacheEnabled
readwritenonatomicassign

It helps to improve scroll performance by "pre-cache" height of cells that have not been displayed on screen. These calculation tasks are collected and performed only when "RunLoop" is in "idle" time.

Auto turn on when you use "-fd_heightForCellWithIdentifier:cacheByIndexPath:configuration".
